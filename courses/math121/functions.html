Functions Activity

This activity reviews two important concepts related to functions: domain/range, and function composition.

The domain of a function is the set of values of the independent variable for which the function is defined. Typically, we choose the subset of the real numbers for which the function is defined. So,

  • Polynomial functions are defined for all of \(\mathbb{R}\)
  • Rational functions are defined for all of \(\mathbb{R}\) except \(x\) which cause the denominator to be zero
  • Even root functions are defined for all of \(\mathbb{R}\) except \(x\) which result in a negative number under an even root (note odd roots are defined for negative \(x\))
  • Sine and Cosine are defined for all of \(\mathbb{R}\)
  • Tangent is defined for all of \(\mathbb{R}\) except \(x = \frac{\pi}{2} + k\pi\), for any integer \(k\)

The range of a function is the set of values the function actually takes on, somewhere in its domain. For example,

  • The function \(f(x) = x^2\) has domain \(\mathbb{R}\), and range \(\mathbb{R}^*\) (this denotes non-negative real numbers), since a square can never be negative
  • The function \(f(x) = \sin(x)\) has domain \(\mathbb{R}\) and range \([-1,1]\), since it oscillates between \(-1\) and \(1\) and never exits that range
  • The function \(f(x) = \sqrt{x+4}-6\) has domain \((-4,\infty)\) and range \((-6,\infty)\)

Domain and range can be more complicated to find for function compositions: obtained by applying multiple functions in succession. The composition of two functions \(f\) and \(g\) is denoted by \[(f \circ g)(x) = f(g(x)) \hspace{1cm} \text{ or } \hspace{1cm} (g \circ f)(x) = g(f(x))\] That is, composition is not commutative: the order in which you apply two functions matters, and generally will result in different functions. For example, where \(f(x) = x+3\) and \(g(x) = x^2\), \[(f \circ g)(x) = f(g(x)) = f(x^2) = x^2+3 \hspace{1cm} \text{ and } \hspace{1cm} (g \circ f)(x) = g(f(x)) = g(x+3) = (x+3)^2 = x^2 + 6x + 9\] To find the domain and range of a more complicated function, say \[h(x) = \sqrt{\frac{1}{x-3}},\] we may find it helpful to decompose it into two functions \(f\) and \(g\) whose composition \(h\). This will be something we do frequently in the study of calculus: being able to recognize compositions of functions which have not been explicitly named as such as crucial to applying the Chain Rule, the differentiation rule applied most frequently. We observe that if \[f(x) = \sqrt{x} \hspace{1cm} \text{ and } \hspace{1cm} g(x) = \frac{1}{x-3}, \hspace{1cm} \text{ then } \hspace{1cm} (f \circ g)(x) = f(g(x)) = f\left(\frac{1}{x-3}\right) = \sqrt{\frac{1}{x-3}} = h(x)\] We have a chain of mappings: \[f:X \rightarrow Y \hspace{1cm} \text{ and } \hspace{1cm} g:Y \rightarrow Z, \hspace{1cm} \text{ so } \hspace{1cm} h:X \rightarrow Z\] To determine the domain of \(h\), we find the subset of the domain of \(g\) for which \(g(x)\) is in the domain of \(f\). The domain of \(g\) is \(\mathbb{R} \setminus \{3\}\) (that is, the real numbers excluding 3, since that would give a zero denominator). Then since \(f\) can't take negative inputs, the domain of \(h\) is the subset of \(\mathbb{R} \setminus \{3\}\) for which \(f(x) = \frac{1}{x-3}\) is nonnegative, namely \((3,\infty)\).
