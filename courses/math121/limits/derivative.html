Limit Definition of the Derivative

The derivative of a function \(f(x)\) at a particular value \(x=a\), denoted by \(f'(a)\), is the slope of the tangent line to \(f\) at \(a\). It can be found using the limit definition \[f'(a) = \lim_{h \to 0}\frac{f(a+h)-f(a)}{h}\] For example, to find the slope of the tangent line to the function \(f(x) = x^2\) at \(x = 3\), we compute \[f'(3) = \lim_{h \to 0}\frac{f(3+h)-f(3)}{h} = \lim_{h \to 0}\frac{(3+h)^2-3^2}{h} = \lim_{h \to 0}\frac{9 + 6h + h^2 - 9}{h} = \lim_{h \to 0}\frac{6h+h^2}{h} = \lim_{h \to 0}(6 + h) = 6 + 0 = 6\] It would be very time-consuming to perform this computation repeatedly if you wanted the slope of the tangent at different points on the same function, so we can instead view the derivative of a function as itself being a function, \[f'(x) = \lim_{h \to 0}\frac{f(x+h)-f(x)}{h}\] We will restrict our attention to relatively "simple" functions' derivatives. Once you learn differentiation rules, this definition becomes obsolete, as they are far quicker to use. Evaluating these limits will involve using the same techniques as in Limits I (such as factoring, rationalizing, or combining fractions) to cancel the \(h\) from the denominator and then do direct substitution.

Suppose we want to find the equation of the tangent line to \(f(x) = \sqrt{2x+5}\) at the point \((2,3)\). We can start by finding the general form of the derivative using the definition: \[f'(x) = \lim_{h \to 0}\frac{\sqrt{2(x+h)+5}-\sqrt{2x+5}}{h} = \lim_{h \to 0} \frac{(\sqrt{2x+2h+5}-\sqrt{2x+5})(\sqrt{2x+2h+5}+\sqrt{2x+5})}{h(\sqrt{2x+2h+5}+\sqrt{2x+5})} =\lim_{h \to 0}\frac{2x+2h+5-(2x+5)}{h(\sqrt{2x+2h+5}+\sqrt{2x+5})}\] \[=\lim_{h \to 0}\frac{2h}{h(\sqrt{2x+2h+5}+\sqrt{2x+5})} =\lim_{h \to 0}\frac{2}{(\sqrt{2x+2h+5}+\sqrt{2x+5})} = \frac{2}{\sqrt{2x+5}+\sqrt{2x+5}} = \frac{2}{2\sqrt{2x+5}} = \frac{1}{\sqrt{2x+5}}\] Then to find the slope of the tangent at \(x=2\): \[f'(2) = \frac{1}{\sqrt{2(2)+5}} = \frac{1}{\sqrt{9}} = \frac{1}{3}\] We have the slope of the tangent line, \(\frac{1}{3}\), and a point on it, \((2,3)\), so we use the point-slope form of a line: \[y - y_1 = m(x - x_1) \implies y - 3 = \frac{1}{3}(x-2)\] (Which you could rearrange into the slope-intercept form if you prefer).
