Logarithms

Logarithms are the inverse functions of exponentials. That is, they swap the places of the independent and dependent variable. They are defined by \[\log_ab= c \iff a^c = b\] So if you want to evaluate \(\log_ab\), you are trying to answer the question "To what power must I raise \(a\) in order to obtain \(b\)?" This question is relatively straightforward for some nice natural numbers: \[\log_28 = 3, \hspace{3mm} \text{because } 2^3 = 8\] and a bit trickier if a fractional exponent is required: \[\log_{27}9 = \frac{2}{3}, \hspace{3mm} \text{because } 27^{\frac{2}{3}} = \left(\sqrt[3]{27}\right)^2 = 3^2 = 9\] We have some "logarithm laws" which are largely analogous to the exponent laws: \[\log_a(bc) = \log_ab + \log_ac, \hspace{18mm} \log_a\left(\frac{b}{c}\right) = \log_ab - \log_ac, \hspace{18mm} \log_a(b^c) = c\log_ab\] Well, the first two are what you'd expect - that rightmost one is different, but a particularly important one. It enables us to move exponents down, which is particularly important when a variable appears in an exponent, which would otherwise be difficult to solve for. Consider, we have strategies for solving equations when the variable appears in the base: \[x^2 + 7x + 12 = 0\] is relatively simple to solve, whereas \[2^{x^2-5x+8} = 4\] should give you pause. But by applying a logarithm to both sides of an equation, we can take advantage of the logarithm laws to bring that variable down. And if we choose the base of our logarithm cleverly, we can ensure that things work out nicely: \[\log_2\left(2^{x^2-5x+8}\right) = \log_24 \implies (x^2-5x+8)\log_22 = 2 \implies x^2-5x+8 = 2 \implies x^2-5x+6 = 0 \implies (x-2)(x-3) = 0 \implies x = 2 \text{ or } 3\]
