Trig Basics Activity

The basic trigonometric functions, Sine, Cosine, and Tangent (denoted \(\sin(x)\), \(\cos(x)\), and \(\tan(x)\)) are functions whose input is an angle of a right triangle, and output is the ratio of the lengths of two sides of that triangle: \[\sin(x) = \frac{\text{opp}}{\text{hyp}}, \hspace{1cm} \cos(x) = \frac{\text{adj}}{\text{hyp}}, \hspace{1cm} \tan(x) = \frac{\text{opp}}{\text{adj}}\] In the past, one would look up values of these functions in cumbersome tables. Now, we have calculators with these functions built-in, and so no longer need the tables. That said, one should be able to sketch the graphs of these functions, and from the graphs easily see the values of any multiple of \(\frac{\pi}{2}\). Note that Sine and Cosine both have period \(2\pi\), and Tangent has period \(\pi\): that is, \(\sin(x) = \sin(x + 2k\pi)\), \(\cos(x) = \cos(x + 2k\pi)\), and \(\tan(x) = \tan(x + k\pi)\) for any integer \(k\).

There are certain "special" values of these functions which we should be able to find easily with the aid of the two "special triangles": \[\sin\left(\frac{\pi}{4}\right) = \cos\left(\frac{\pi}{4}\right) = \frac{1}{\sqrt{2}}, \tan\left(\frac{\pi}{4}\right) = 1, \sin\left(\frac{\pi}{3}\right) = \cos\left(\frac{\pi}{6}\right) = \frac{\sqrt{3}}{2}, \sin\left(\frac{\pi}{6}\right) = \cos\left(\frac{\pi}{3}\right) = \frac{1}{2}, \tan\left(\frac{\pi}{3}\right) = \sqrt{3}, \tan\left(\frac{\pi}{6}\right) = \frac{1}{\sqrt{3}}\]

A useful theorem for finding the lengths of sides of triangles is the Pythagorean Theorem: \[a^2 + b^2 = c^2\] for any right triangle whose hypotenuse (the longest side, opposite the right angle) has length \(c\), and whose legs (the other two sides) have lengths \(a\) and \(b\). So, if we know two side lengths of a right triangle, we are able to find the length of the remaining side. Suppose we have a right triangle whose hypotenuse has length \(5\), and one of its legs (without loss of generality, name it \(a\)) has side length \(3\). Then \[a^2 + b^2 = c^2 \implies 3^2 + b^2 = 5^2 \implies 9 + b^2 = 25 \implies b^2 = 16 \implies b = 4\] When all three side lengths are integers, we have what is called a Pythagorean triple: \((3,4,5)\).
