Resources/Links

Useful Math Resources

A source of funny (and educational) math comics:
https://mathwithbaddrawings.com/

A helpful site for learning many things in math you feel the need to brush up on (at first glance it may appear the target audience is children, but it covers from the basics of arithmetic all the way up to some differential equations!):
https://www.mathsisfun.com/

Great for computation and visualization (I use it frequently):
https://www.wolframalpha.com/

Also useful for visualization:
https://www.desmos.com/

The mathematical computation software I use to generate many figures, particularly great at matrix algebra (a typical high-level programming language):
https://www.mathworks.com/products/matlab.html

Sage is a somewhat more difficult mathematics software to learn to use, but is open-source and powerful:
https://www.sagemath.org/
